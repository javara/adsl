# ADSL Website

This project is a small static website about ADSL, originally created as a school assignment and later adapted for the web.

## Contents

- Information about ADSL technology
- Pages for introduction, requirements, advantages, models, tariffs, and more
- Links, contact page, and glossary

## Project Structure

- `index.htm` — main entry page
- `index-adsl.htm` — ADSL content index
- `*.htm` — individual content pages
- `styles.css` — shared stylesheet
- `images/` — site images

## How to Use

1. Open `index.htm` in your browser, or
2. Serve the folder with a simple local web server.

## Notes

- The site uses static HTML files.
- Styling is centralized in `styles.css`.
- This project is intended for educational/demo purposes.
